    ⚡ FASTEST DEPLOYMENT FROM ANDROID:


    ═══════════════════════════════════════════════════════════════
    📋 COPY-PASTE THESE COMMANDS IN TERMUX:
    ═══════════════════════════════════════════════════════════════

    STEP 1: Setup Termux (One time only)
    ────────────────────────────────────────────────────────────

    pkg update && pkg upgrade -y
    pkg install nodejs-lts git -y


    STEP 2: Navigate to your project
    ────────────────────────────────────────────────────────────

    cd ~/your-project-folder

    (Replace with your actual folder name!)


    STEP 3: Clean up file structure
    ────────────────────────────────────────────────────────────

    bash cleanup.sh

    (Or manually run the commands from TERMUX-DEPLOY.md Step 3)


    STEP 4: Create .env file
    ────────────────────────────────────────────────────────────

    nano .env

    Add these lines (with your actual Supabase values):
    VITE_SUPABASE_URL=https://your-project.supabase.co
    VITE_SUPABASE_ANON_KEY=your-key-here

    Press Ctrl+X (Volume Down + X) to save
    Press Y, then Enter


    STEP 5: Build
    ────────────────────────────────────────────────────────────

    npm install
    npm run build


    STEP 6: Deploy to Vercel
    ────────────────────────────────────────────────────────────

    npm install -g vercel
    vercel login
    vercel

    When asked:
    • Most questions: Press ENTER
    • Override settings? → y
    • Build Command? → npm run build
    • Output directory? → dist


    STEP 7: Add env vars in Vercel
    ────────────────────────────────────────────────────────────

    In Chrome browser:
    1. Go to vercel.com/dashboard
    2. Your project → Settings → Environment Variables
    3. Add: VITE_SUPABASE_URL (value from .env)
    4. Add: VITE_SUPABASE_ANON_KEY (value from .env)
    5. Select "All" for environments


    STEP 8: Final deploy
    ────────────────────────────────────────────────────────────

    vercel --prod


    ✅ DONE! YOUR APP IS LIVE! 🎉
